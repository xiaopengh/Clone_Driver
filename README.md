| Mutation Type                    | Description                                                                  | Molecular Change                                                                                                      | Effect on Protein                                                                                                                                             | Example                                                                                                     |
| -------------------------------- | ---------------------------------------------------------------------------- | --------------------------------------------------------------------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------- | ----------------------------------------------------------------------------------------------------------- |
| **Point mutation**               | A change of a single nucleotide at one position in the DNA.                  | One base substituted for another (e.g. A→G, C→T).                                                                     | Depends on resulting codon: may be silent, missense, or nonsense.                                                                                             | Change of A→G in codon GCA → GCG still codes for Ala (silent).                                      |
| **Missense mutation (错义突变)**     | A point mutation that alters a codon so it specifies a different amino acid. | Single‐base substitution that changes one codon to another coding for a different residue.                            | Incorporates wrong amino acid—can impair, enhance, or have little effect.                                                                                     | GAG → GTG changes Glu → Val; causes sickle‐cell disease (β‐globin).                                     |
| **Nonsense mutation (无义突变)**     | A point mutation that converts a codon for an amino acid into a stop codon.  | Single‐base substitution creating one of the three stop codons (UAA, UAG, UGA).                                       | Premature truncation of the polypeptide; usually loss of function.                                                                                            | TAC → TAG changes Tyr → Stop; yields truncated protein.                                                 |
| **Insertion / Deletion (indel)** | Addition or loss of one or more nucleotides in the DNA sequence.             | − **In‐frame indel**: insertion/deletion in multiples of 3 bp (codon(s))− **Frameshift**: indel not divisible by 3 bp | − In‐frame: adds/removes whole codon(s) → extra or missing residue(s).− Frameshift: alters downstream reading frame → widespread missense + often early stop. | **Frameshift**: deletion of one “C” in …CC\*\*C\*\*A… → …CCA… shifts frame and corrupts all downstream AAs. |
